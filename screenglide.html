Баннер с расхлопом

Подготовить баннер этого типа можно воспользовавшись одним из наших шаблонов, которые покрывают наиболее распространённые сценарии (одна кликабельная область и расхлоп по наведению; несколько кликабельных областей и расхлоп по наведению; одна кликабельная область и расхлоп/схлоп по клику; несколько кликабельных областей и расхлоп/схлоп по клику).

Подготовка баннера

Первые два шага, добавление instance name и шаблона, являются общими для всех случаев.

  1. Подготовьте в Adobe Animate CC креатив
    На разных слоях разместите два symbol, которые будут соответствовать баннеру в схлопнутом и расхлопнутом состояниях, если нужна кнопка "закрыть", то добавьте её в movie clip расхлопнутого состояния
    Так же задайте им instance names.
    Лучше использовать instance names "expanded", "collapsed" и "closed" для расхлопнутого, схлопнутого состояния и кнопки "закрыть" соответственно. Тогда можно без изменений пренести примеры кодов из этой инструкции в Animate.
    Подробнее про instance names можно почитать в соответствующей инструкции

  2. Подключите нужный шаблон ADFOX к проекту.

    • Одна кликабельная область и расхлоп по наведению: Скачать
    • Несколько кликабельных областей и расхлоп по наведению: Скачать
    • Одна кликабельная область и расхлоп/схлоп по клику: Скачать
    • Несколько кликабельных областей и расхлоп/схлоп по клику: Скачать

  3. В зависимости от логики работы баннера добавьте один из примеров кода в слой Actions.
    Коды приведены ниже.

    Одна кликабельная область и расхлоп по наведению

    //Поменяйте instance name если требуется
    expanded = this.expanded;
    collapsed = this.collapsed;
    setAdfox();

    Несколько кнопок и расхлоп по наведению

    //Поменяйте instance name если требуется
    expanded = this.expanded;
    collapsed = this.collapsed;
    window.buttons.push(
    //Пропишите через запятую пути кнопок, добавив вначале this
    this.expanded.button1, this.expanded.button2, this.expanded.button3, this.expanded.button4
    //Конец места для кнопок
    );
    setAdfox();

    Одна кликабельная область и расхлоп/схлоп по клику

    //Поменяйте instance name если требуется
    expanded = this.expanded;
    collapsed = this.collapsed;
    close = this.expanded.close;
    setAdfox();

    Несколько кнопок и расхлоп/схлоп по клику

    //Поменяйте instance name если требуется
    collapsed = this.collapsed;
    expanded = this.expanded;
    close = this.expanded.close;
    window.buttons.push(
    //Пропишите через запятую пути кнопок, добавив вначале this
    this.expanded.button1, this.expanded.button2
    //Конец места для кнопок
    );
    setAdfox();

  4. После добавления кода в Actions опубликуйте проект.

  5. Если вы используете вариант с несколькими кнопками, то прописывать ссылки нужно на вкладке "События" в параметрах баннера.
    Первый instance в строке кода будет соответствовать полю "Событие 1" в параметрах баннера. Название события в интерфейсе может быть любое, в этом примере я прописал instance names для наглядности их соответствия с ссылками.

    this.expanded.button1, this.expanded.button2
